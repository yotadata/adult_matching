# 利用規約のポイント

エンドユーザー向けに、最初に伝えておくべき利用規約の要点をまとめます。新規登録モーダル・このサイトについてページでは、以下のサマリーを案内したうえで詳細版（Supabase 上の `terms` テーブル or CMS）へリンクしてください。

1. **対象年齢**: 性癖ラボは 18 歳以上のみ利用可能です。登録時には年齢確認チェックボックスへの同意を必須とします。
2. **利用目的と登録理由**: AI が嗜好を学習し、スワイプ結果やレコメンドを最適化するためにアカウント登録が必要です。メールアドレス形式ではない ID でも構いませんが、好みを保存するための識別子は必須です。
3. **取得しない情報**: 個人を特定できる氏名・住所・クレジットカード情報などのセンシティブデータは取得しません。
4. **コンテンツ取り扱い**: 本サービスで提供する作品データ・サムネイル・テキストのスクレイピングや自動取得を禁止します。検知した場合はアカウント停止や法的措置を行う場合があります。
5. **カスタマーサポート**: 利用停止や利用規約違反への対応、アカウント削除の申し出はアカウント設定ページのモーダル経由で案内する Google Form から受け付けます。送信時にはユーザー ID の入力が必要です。

運用で新しい禁止事項や免責が増えた場合は、最初にこのファイルを更新し、該当ページの文言へ反映してください。
