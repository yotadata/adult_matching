# Google インデックス方針

Google などのクローラにどのページをインデックスさせるかを整理します。基本方針は以下です。

- **公開用の情報ページ / 共有リンクのみインデックスを許可**  
  例: スワイプ画面（`/swipe`）、このサイトについて（`/about`）など。
- **ログイン必須 / 会員向け画面はインデックス禁止**  
  例: `/search`, `/swipe`, `/lists`, `/insights` など。
- API (`/api/*`) や内部確認用のページはすべてクローラ拒否。

## ページ一覧

| パス | 内容 | インデックス可否 |
| --- | --- | --- |
| `/` | `/search` へのリダイレクト（アプリ入口） | ❌ |
| `/swipe` | スワイプ UI（トップページとして公開） | ✅ |
| `/about` | サイトについての情報ページ | ✅ |
| `/search` | 「AIで探す」画面（要ログイン） | ❌ |
| `/lists` | 会員用のマイリスト | ❌ |
| `/insights` | 好み分析ダッシュボード | ❌ |
| `/api/*` | API エンドポイント | ❌ |

## robots.txt

`frontend/public/robots.txt` で上記ポリシーを反映しています。Next.js がビルド時に `/robots.txt` を公開するため、追加の設定は不要です。
