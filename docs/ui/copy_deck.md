## 運用メモ

1. 本表で「提案」となっている箇所は、UI 実装時に変更する前に PM/デザイナーの承認を得る。
2. 新規画面の文言はこの Copy Deck に追記してから実装し、`docs/ui/glossary.md` と整合を取る。
3. 多言語対応時は英語版の column を追加し、簡潔な翻訳キーを併記する。
